# Godot Pixel Painter

[English README](./README.en.md)

一个基于 **Godot 4.6** 的像素绘制与精修工具，支持单图像素化、序列帧编辑、透明规则处理、限色分析与整图导出。

## 主要功能

- 像素画布编辑：支持 `16x16` 到 `64x64` 画布、左键上色、右键擦除。
- 预设调色板：内置多组常用像素风调色板，可快速切换。
- 参考图像素化：导入图片后自动量化到当前调色板。
- 智能推荐调色板：自动评估误差并切换到更匹配的预设。
- 图像像素分析：生成动态调色板 `此图像素`，可用于快速限色。
- 限色与融合：通过 `限色 + 宽容度` 控制颜色数量和融合强度。
- 透明规则面板：可选透明色、设置透明宽容、并应用到单帧或所有帧。
- 序列帧工作流：导入 sprite sheet、网格切分、底部预览、逐帧编辑。
- 帧偏移工具：通过右上角 `偏` 进入偏移模式，拖拽后应用到目标帧。
- 导出能力：导出当前画布 PNG、导出配置 JSON、`生` 按钮生成整图（导出时可自选文件夹）。

## 操作介绍

### 基础操作

- 左键：绘制当前选中颜色。
- 右键：擦除（设为透明）。
- 中键拖拽：平移画布。
- 滚轮：缩放画布。
- 数字键 `1-9`：切换前 9 个调色板颜色。
- `C`：清空画布（可撤销）。
- `Ctrl+Z`：撤销最近一次操作。
- `E`：导出当前画布 PNG（弹出目录选择）。

### 顶部工具栏

- `画布`：切换画布尺寸。
- `调色板`：切换预设调色板。
- `清空`：清空当前画布。
- `导出PNG`：导出当前画布（弹出目录选择）。
- `导出配置 / 导入配置`：保存或恢复画布、调色板和像素数据。
- `还原`：恢复当前来源图（单图或当前帧）的原始状态。
- `推荐调色板`：按当前来源图自动推荐预设。
- `分析图像像素`：分析当前来源图并生成 `此图像素` 调色板。
- `限色`：限制动态调色板颜色数量。
- `最大`：一键恢复到当前可用最大颜色数。
- `宽容度`：控制颜色融合强度（`0-100`）。

### 右上角快捷按钮

- `EN/中`：界面语言切换。
- `帧`：导入序列帧图片并进入帧分割流程。
- `偏`：进入帧偏移模式；再次点击（按钮变为“应用偏移”）应用偏移。
- `生`：将当前预览帧拼接为整图并导出 PNG（弹出目录选择）。

### 底部帧预览

- 左键某帧：加载到主画布编辑。
- 右键某帧：打开当前帧框选微调模式。
- 键盘 `←/→`：切换当前选中帧。

### 透明操作面板

- 在左侧调色板上右键颜色：添加/移除透明基准色。
- `透明宽容`：扩展相近颜色进入透明集合。
- 右键透明色块：从透明集合中移除该颜色。
- `应用至所有帧`：把当前透明规则批量应用到全部帧。

## 典型流程

### 单图像素化流程

1. 导入参考图。
2. 点击 `推荐调色板`（可选）。
3. 点击 `分析图像像素`，再通过 `限色 / 宽容度` 调整风格。
4. 手工微调像素后导出 PNG。

### 序列帧流程

1. 点击右上角 `帧` 导入 sprite sheet。
2. 在帧分割窗口设置行列，确认帧区域。
3. 在底部预览条切帧并逐帧编辑。
4. 使用透明规则与偏移工具做批量或精修处理。
5. 点击右上角 `生` 导出拼接整图。

## 输出与文件

- PNG 输出目录：每次导出时由用户在系统目录选择框中指定。
- 配置文件：`res://configs/pixel_canvas_config.json`
- 主场景：`res://scenes/pixel_painter.tscn`
- 核心脚本：`res://scripts/pixel_painter.gd`

配置文件关键字段：

- `grid`：画布宽高与单元格尺寸。
- `palette`：当前调色板（HEX）。
- `pixels`：二维像素索引（`-1` 表示透明）。
- `meta.preset_name`：当前预设调色板名称。

## 运行方式

1. 使用 Godot `4.6` 打开项目目录。
2. 运行主场景 `res://scenes/pixel_painter.tscn`（项目已默认配置）。
