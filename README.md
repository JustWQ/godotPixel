# Godot Pixel Painter

[English README](./README.en.md)

一个基于 **Godot 4** 的像素画小工具，支持画布绘制、预设调色板、参考图像素化、自动推荐调色板、图像颜色分析与限色融合。
特别适合用于 AI 生成像素图的后处理与风格统一。

适合用于：
- 像素角色草图
- 参考图快速转像素风
- 调色板实验与限色练习

## 功能特性

- 多画布尺寸：`16x16` 到 `64x64`
- 预设调色板：内置多组复古/主题配色
- 多语言 UI：右上角一键切换中/英
- 选择图片像素化：通过系统文件选择框导入并自动量化到当前调色板
- 调色板推荐：自动评估各预设误差并切换最优
- 图像像素分析：生成动态调色板 `此图像素`
- 限色控制：可输入颜色数量上限（不超过图像颜色种类），支持一键 `最大`
- 颜色宽容度（`0-100`）：相似颜色融合，自动重映射像素
- 序列帧工作流：导入 sprite sheet，网格切分、底部帧预览、左右键切帧
- 帧级偏移：在主画布直接拖拽绿色网格叠层，应用后替换原帧
- 整图生成：将预览帧按近似正方形网格拼接并导出
- 画布视图控制：中键拖拽平移、滚轮缩放
- 导出 PNG：输出到 `res://output/`
- 配置导入导出：`JSON` 保存/恢复画布、调色板和像素数据

## 运行环境

- Godot `4.6`（项目配置特性为 `4.6`）
- Windows/macOS/Linux（编辑器可运行）

## 快速开始

1. 克隆仓库：

```bash
git clone https://github.com/<your-name>/godot-pixel.git
cd godot-pixel
```

2. 使用 Godot 4.6 打开项目目录。
3. 运行主场景（已配置为 `res://scenes/pixel_painter.tscn`）。

## 使用说明

### 基础绘制

- 左键：上色
- 右键：擦除
- 数字键 `1-9`：快速切换前 9 个调色板颜色
- 方向键 `←/→`：切换底部选中预览帧
- `C`：清空画布
- `E`：导出 PNG

### 大画布查看

- 中键拖动：平移画布
- 鼠标滚轮：缩放画布（以鼠标位置为中心）

### 参考图工作流

1. 点击 `选择图片像素化`，打开系统文件选择框导入图片（会自动像素化到当前调色板）。
2. 点击 `推荐调色板`，自动切换到误差更低的预设。

### “此图像素”与限色

1. 点击 `分析图像像素`，生成动态调色板 `此图像素`。
2. 使用 `限色` 输入框减少颜色数量。
3. 调整 `宽容度(0-100)` 控制近似颜色融合强度：
- `0`：仅相同颜色，不做近似融合
- `100`：高强度融合，颜色种类会显著减少

减少颜色后，系统会对被移除颜色进行最近似补色并自动更新画布。

### 序列帧工作流

1. 点击右上角 `帧` 按钮导入序列图（sprite sheet）。
2. 在帧分割窗口调整 `行/列`，绿色网格会实时跟随变化。
3. 底部出现横向帧预览：
- 左键：加载该帧到编辑画布
- 右键：进入当前帧的框选微调模式
4. 选中帧后，`推荐调色板`、`分析图像像素`、`宽容度/限色` 会基于该帧实际区域计算。

### 帧偏移（画布偏移语义）

1. 在底部先选中一个帧。
2. 点击右上角 `偏` 进入偏移模式（按钮文本会变为 `应用偏移`）。
3. 在主画布拖拽绿色半透明网格画布（按网格吸附）。
4. 再次点击按钮应用：将旧内容“投影”到新画布位置，并替换原帧。

### 生成整图

- 点击 `生成整图`，会将当前预览帧按近似正方形网格拼接成一张 PNG，导出到 `res://output/`。

## 项目结构

```text
godot-pixel/
├─ scenes/
│  └─ pixel_painter.tscn          # 主场景
├─ scripts/
│  └─ pixel_painter.gd            # 核心逻辑（UI、绘制、导入导出、算法）
├─ configs/
│  ├─ pixel_canvas_config.json    # 配置文件（画布/调色板/像素）
│  └─ reference.png               # 默认参考图
├─ output/                        # PNG 导出目录
└─ project.godot                  # Godot 项目配置
```

## 配置文件说明

配置文件路径：`res://configs/pixel_canvas_config.json`

主要字段：
- `grid`: 画布宽高、单元格尺寸
- `palette`: 当前调色板（HEX）
- `pixels`: 二维像素索引（`-1` 表示透明/空）
- `meta.preset_name`: 当前预设名称
